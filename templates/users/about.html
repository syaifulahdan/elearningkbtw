Tentang Tugas

    Tentang Tugas

    ini adalah Proyek Akhir matakuliah KBTW - Komputasi Bergerak dan Teknologi Web, project akhir yang saya bahas tentang E-learning, semoga dengan tugas ini dapat berbagi pengalaman dan sharing pengetahuan tentang bagaimana membangun web elearning berbasis python dengan menggunakan framework django

    Fasilitas Apa Saja Yang Ada di Web ini

    di website ini memiliki fasilitas mulai dari Pendaftaran user, Proses pembuatan matakuliah, Upload Recource teks, video dan berkas

    Apa saja yang dibutuhkan

    : Virtual Environtmen, Python, Django 1.9 Framework, Bootstrap-css, jquery-2, Sqlman database

Bagaimana Cara Menggunakannya

  • 1.
    Login Sebagai Mahasiswa

    gunakan user : mahasiswa, password : mahasiswa ( Login sebagai mahasiswa dapat mengakses mata kuliah yang sudah disajikan ke oleh dosen, Klik Daftar Matakuliah)

    Login Sebagai Dosen.

    gunakan user : Dosen, password : dosen ( untuk membuat matakuliah pilih > DAFTAR MATAKULIAH . KEMUDIAN KLIK MENU DOSEN)
